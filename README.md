# Pingui Marmolería 🪨

**Arte en Piedra Natural — Punta Arenas, Chile**

Sitio web de Marmolería Pingui, especialistas en libros de mármol, placas conmemorativas, crucifijos y arte funerario en piedra natural.

## 🚀 Tecnologías

- HTML5 semántico + CSS3 + JavaScript vanilla
- Google Fonts (Inter + Cormorant Garamond)
- Font Awesome 6
- FormSubmit.co para el formulario de cotización
- Unsplash para imágenes de galería
- Desplegado en GitHub Pages

## ✨ Características

- ✅ Diseño responsive optimizado para móviles
- ✅ Fondo tipo mármol generado con SVG y filtros feTurbulence
- ✅ Lightbox para ampliar imágenes de productos
- ✅ Formulario de cotización funcional con validación
- ✅ WhatsApp flotante con animación pulse
- ✅ SEO con Open Graph, Twitter Cards y JSON-LD
- ✅ Animaciones scroll-reveal con Intersection Observer
- ✅ Contador de visitas (localStorage)
- ✅ Accesibilidad y etiquetas ARIA
- ✅ Soporte para `prefers-reduced-motion`
